# Designlang

A tool for design extraction and analysis.

## Quick Start

### Use directly (no install needed)
```bash
npx designlang https://example.com
```

### Install globally
```bash
npm install -g designlang
```

### Use as an agent skill (Claude Code, Cursor, Codex, 40+ agents)
```bash
npx skills add Manavarya09/design-extract
```

## About

Designlang is available for use with popular AI agents and development tools, making it easy to integrate design extraction capabilities into your workflow.
